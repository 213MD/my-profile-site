## 네이밍 컨벤션

- **컴포넌트/클래스**: PascalCase

## 타입 규칙
**금지**: any 타입 사용 불가
- 모든 변수와 함수에 명확한 타입 지정 필수

**Why:** 타입 안정성 강화, IDE 자동완성 및 버그 조기 발견
**How to apply:** 변수 선언 시 타입 주석 추가, 함수 매개변수/반환값 명시

## 컴포넌트 구조

- 관심사의 분리: 로직과 뷰 분리
